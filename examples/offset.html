Slideshow Demo (with offset)

This demonstrates how to use a Backstretch slideshow and offsetting the image. In this case, the image should fill and "stick" from the bottom right of the page (or element if used).

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script src="jquery.backstretch.min.js"></script>
<script>
    $.backstretch([
      "dome.jpg"
      ], {
        offsetX: 0.9,
        offsetY: 0.9
    });
</script>
